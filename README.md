## Expo/React Native test app

To experiment with some techniques used in a separate commerical project

### Techniques

- Nested navigators
- Audio recording
- Persistence of application state
- oauth2 token
- Google Map API

### Software versions

Tested on Mac OS X 10.13.16

- Node.js 8.11.3
- NPM 5.6.0
- expo-cli 1.1.0-beta4
- yarn 1.7.0
